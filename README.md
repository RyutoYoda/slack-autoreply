# Slack AI Auto-Reply

Slackでメンションされた時に、OpenAI APIを使って自動で返信を生成するChrome拡張機能

## 機能

- **メンション検知**: 自分へのメンション(@)を自動検知
- **AI返信生成**: OpenAI APIで会話の文脈を理解して適切な返信を生成
- **半自動/完全自動モード**:
  - 半自動: 返信文を入力欄に生成し、送信は手動で確認
  - 完全自動: 返信文を自動生成・送信

## インストール方法

1. このリポジトリをクローン、またはZIPでダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

### 1. OpenAI APIキーの設定

1. 拡張機能のポップアップを開く
2. 「OpenAI APIキー設定」を展開
3. APIキーを入力して「保存」ボタンをクリック

### 2. 自動返信の有効化

1. Slackを開く
2. 拡張機能のポップアップを開く
3. 「自動返信機能」をオンにする
4. 「自動送信」のオン/オフを選択:
   - **オン**: 返信を自動生成・送信
   - **オフ**: 返信を入力欄に生成するだけ（送信は手動）

### 3. 動作

- 誰かがあなたにメンション(@yourname)すると、AIが自動で返信を生成します
- 半自動モードの場合、入力欄に返信が表示されるので、内容を確認してから送信ボタンを押してください
- 完全自動モードの場合、自動で送信されます

## 注意事項

- OpenAI APIキーが必要です（有料）
- APIの使用には料金が発生します
- Slackのページを開いている間のみ動作します
- SlackのDOM構造変更により動作しなくなる可能性があります
- テスト環境で十分に動作確認してから本番環境で使用してください

## ファイル構成

```
├── manifest.json   # 拡張機能の設定
├── popup.html      # ポップアップUI
├── popup.js        # ポップアップのロジック
├── content.js      # メイン処理（メンション検知・返信生成）
└── background.js   # OpenAI API呼び出し
```

## トラブルシューティング

### 返信が生成されない

- OpenAI APIキーが正しく設定されているか確認
- ブラウザのコンソールでエラーメッセージを確認（`SLE:` でフィルタ）
- Slackのページを再読み込み

### 自動送信が動作しない

- Slackの送信ボタンが表示されているか確認
- 半自動モード（自動送信オフ）になっていないか確認
